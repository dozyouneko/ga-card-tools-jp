このモックが表す挙動（v2で更新）

  • 構成順: マテリアル → メイン → サイド → 合計 → フォーマット適合
  • 合計 = マテリアル+メイン+サイドの全カードを合算（「検討中」は除外）
  • 表記は「英語（日本語訳）」 — 既存の label() 関数の出力形式そのまま（例: WIND（風））
  • Floating Memoryはカードの効果テキスト(英語)への文字列一致で検出。[Class Bonus]/[Level N+]等の条件タグ付きは「うち条件付き n枚」と内数表示。0枚のゾーンでは節ごと非表示
  • 提案: フォーマット適合が最下部に移ったため、禁止カードがある時だけ統計最上部に⚠️バナーを表示（このモックでは表示例として常時表示）
  • タブ切替はゾーン表示と統計を丸ごと切替。検索バーは統計中も表示（カード追加で即時更新）
  • 共有閲覧画面にも同じタブバー＋統計パネル（構成同一・編集操作なし）
  • 0枚の分類は表示しない。リロード時はデッキタブに戻る

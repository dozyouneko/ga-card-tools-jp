## 設計からの差分（3件）

1. **`#ev-empty` の出し分けに条件を1つ足した**
   設計は `emptyEl.hidden = n !== 0;` だが、実装は `n !== 0 || rows.length === 0` とした。
   収録0件のときは表の中に「収録済みの大会がありません。」が出るため、
   条件を変えようのない状況で「条件を変えてお試しください。」が重ねて出るのを避けた
2. **`.filter-hits` の margin を `10px 0 0` → `0 0 10px` に変えた**
   要素が絞り込み枠の末尾から表の直上へ移ったため。上マージンのままだと表との間が空いて
   「この表が何件か」の対応が弱くなる
3. **`data/tournaments/index.json` を `git checkout` で戻した**
   `--no-scan` でも `writeIndexJson()` が `updatedAt` を現在時刻で書き直す（`scan`・`events` は不変）。
   検証項目15の趣旨（日次cronの差分と混ざらない）に沿って、タイムスタンプだけの変更はコミットに含めていない

挙動メモ(実装時の参考)

  1. 詳細ダイアログ: 本モックでは実物の `shared/js/card-detail.js` をそのまま読み込んで動かしている(下のscriptタグ参照)。実装時もこのまま流用できる(既存の共有デッキ画面と同じ組み込み方)。
  2. スマホ表示: `shared/css/card-detail.css` の `@media (max-width: 640px)` で自動的に全画面シート表示になる(既存の仕様。追加対応不要)。
  3. タイルの見た目: 共有デッキ画面(`renderDeckView()`・`.cardph`/`.view-grid`)と同一。
  4. イラストの版選び: 既定は先頭の版(`editions[0]`)。「最新版を優先」等が必要になれば`cardImages()`で全版取得し並べ替えるだけで対応可。
  5. 未解決カード: `.noimg`(スラグ解決できなかった場合の代替表示)も既存スタイルをそのまま使う。
  6. 枚数超過・禁止カード: 大会時点のフォーマット適合は問わない(過去のレギュレーションでの入賞デッキのため、現行ルールでの警告表示はしない方針)。

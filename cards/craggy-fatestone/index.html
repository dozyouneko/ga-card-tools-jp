岩肌の運命石Craggy Fatestone

日本語訳あり
タイプITEM（アイテム）
クラスTAMER（テイマー）
エレメントNORM（ノーム）
サブタイプTAMER（テイマー）、FATESTONE（運命石）
リザーブコスト4

効果（日本語）

対戦相手がメモリーコスト0のカードをマテリアライズするたび、「岩肌の運命石」にバフカウンターを1個置く。

[グオ・ジアボーナス] [レスト]：「岩肌の運命石」をトランスフォームする。この能力は、「岩肌の運命石」にバフカウンターが2個以上置かれている場合にのみ起動する。

効果（英語原文）

Whenever an opponent materializes a card with memory cost 0, put a buff counter on Craggy Fatestone.

[Guo Jia Bonus] [REST]: Transform Craggy Fatestone. Activate this ability only if there are two or more buff counters on Craggy Fatestone.

スペルシュラウド （スペルシュラウドを持つユニットはスペルの対象にできない。）

効果（英語原文）

Spellshroud (Units with spellshroud can’t be targeted by Spells.)

Your opponents can't materialize cards with memory cost 0.

このカードの用語

  • マテリアライズマテリアルデッキのカードを、そのコストを支払って場に出す行為。
  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • バフカウンターアライは自身に置かれたバフカウンター1個につき+1パワーと+1ライフを得る。
  • スペルシュラウド（Spellshroud）このオブジェクトはスペルの対象にできない。

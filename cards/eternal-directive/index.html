Command Automaton (An Automaton ally you control performs this attack.)

フレーバーテキスト

"Your program will never override our will."

このカードの用語

  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • バフカウンターアライは自身に置かれたバフカウンター1個につき+1パワーと+1ライフを得る。
  • 攻撃時（On Attack）このユニットが攻撃を行ったときに誘発する効果。

公式裁定（2件）

※裁定は英語原文です。

2025-05-28

If the ally rested to pay for the cost of the Command card dies before the activation resolves, the Command card will fizzle and not start a combat phase.

2025-05-28

The eligible ally that was rested to pay for activating the Command card is the one that will declare the attack upon the card's resolution.

大海の祝福Ocean's Blessing

日本語訳あり

クラスTAMER（テイマー）
エレメントWATER（水）
サブタイプTAMER（テイマー）、SPELL（呪文）
リザーブコスト2

効果（日本語）

アライリンク （このオブジェクトは、対象のアライにリンクした状態で場に出る。リンクが解除された場合、このオブジェクトをサクリファイスする。）

リンクされたアライは+1[ライフ]を得て、タウントを持つ。

[クラスボーナス] フローティングメモリー （メモリーコストの支払い中に、このカードを墓地から追放してそのコストの1点分の支払いに充ててよい。）

効果（英語原文）

Ally Link (This object enters the field linked to target ally. If the link is broken, sacrifice this object.)

Linked ally gets +1 [LIFE] and has taunt.

[Class Bonus] Floating Memory

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。
  • サクリファイス（Sacrifice）自分のオブジェクトをコストとして墓地へ送る行為。
  • タウント（taunt）アウェイクの間、相手の攻撃宣言時に、可能なら他のオブジェクトより先に対象にされなければならない。ステルスなどにより対象に取れない場合、他のオブジェクトを対象に取ることができる。
  • アライリンク（Ally Link）登場時に自分のアライ1体とリンクし、そのアライを参照する効果を持つファンタジア。
  • リンク（Link）対象のオブジェクトをリンク先として場に出る能力。リンク先を失うと、このオブジェクトはサクリファイスされる。リンク先の種類に応じて「ユニットリンク」「チャンピオンリンク」「ウェポンリンク」などと表記される。

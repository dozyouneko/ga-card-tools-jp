効果（英語原文）

True Sight (This attack can target units with stealth.)

[Class Bonus] On Kill: Put a preparation counter on your champion. (Apply this effect only if your champion’s class matches this card’s class.)

このカードの用語

  • プレパレーションカウンター（preparation counter）一部の能力や起動時に使用するカウンター。主に「プレパレーションN」などの効果によって使用される。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • トゥルーサイト（True Sight）この武器を使う攻撃は、ステルスを持つユニットも対象にできる。
  • ステルス（Stealth）このオブジェクトは、トゥルーサイトを持つ攻撃でない限り、攻撃の対象にならない。
  • キル時（On Kill）このユニットが与えた戦闘ダメージによって、ユニットがキルされた（死亡した）ときに誘発する効果の見出し。ユニット以外の破壊では誘発しない。

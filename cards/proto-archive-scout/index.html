効果（日本語）

「プロトアーカイブの斥候」はアウェイクであるかぎりステルスを持つ。

あなたが「プロトキー・クレスト」という名前のオブジェクトをコントロールしているかぎり、「プロトアーカイブの斥候」は＋1パワーを得る。

効果（英語原文）

Proto Archive Scout has stealth as long as it’s awake.

As long as you control an object named Proto Key Crest, Proto Archive Scout gets +1[POWER].

このカードの用語

  • ステルス（Stealth）このオブジェクトは、トゥルーサイトを持つ攻撃でない限り、攻撃の対象にならない。

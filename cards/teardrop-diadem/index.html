涙滴の宝冠Teardrop Diadem

効果（日本語）

（エグザルテッド → 他のアドバンスドエレメントが有効である限り、このエレメントもあなたにとって有効になる。）

このカードのメモリーコストは、フローティングメモリーのみを使って支払う。

このカードを追放する： カードを3枚あなたのメモリーに引く。

効果（英語原文）

(Exalted — This element is enabled for you as long as you have another advanced element enabled.)

Pay only using floating memory for this card’s memory cost.

Banish Teardrop Diadem: Draw three cards into your memory.

このカードの用語

  • 追放（Banish）カードをゲームから取り除く一時的・恒久的な領域。
  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。
  • エグザルテッド（Exalted）他のアドバンスドエレメントを有効にしている限り、このエレメントも有効になる。
  • アドバンスドエレメント（advanced element）基本エレメント（ノーム・火・水・風）を除くエレメント（ラクサム・アンブラ・テラ・アストラ・クラックス・アーケイン・エクシア・ネオス・エグザルテッド）の総称。

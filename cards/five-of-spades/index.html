効果（日本語）

カーディストリ — (5)：ターン終了時まで「スペードの5」は＋5パワーを得る。この能力は、あなたがコントロールする異なるリザーブコストを持つスートオブジェクト1つにつき、起動コストが(1)少なくなる。この能力は1回のみ起動する。

フローティングメモリー

効果（英語原文）

Cardistry — (5): Five of Spades gets +5[POWER] until end of turn. This ability costs (1) less to activate for each Suited object you control with different reserve costs. Activate this ability only once.

Floating Memory

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • リザーブコストの支払いとして、手札からメモリー領域へ移すこと。リザーブコストはこの方法で支払う。
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。
  • カーディストリ（Cardistry）スート系のオブジェクトの数だけコストが下がる起動型能力。1回のみ起動できる。

公式裁定（1件）

※裁定は英語原文です。

2025-06-27

The "activate this ability only once" applies to each instance of a Cardistry ability on a card. If you have multiple copies of a card with the same Cardistry ability in play, each one can activate its ability one time.

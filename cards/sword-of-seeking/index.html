効果（日本語）

［クラスボーナス］トゥルーサイト （この武器を使う攻撃はステルスを持つユニットを対象にできる。この効果は、あなたのチャンピオンのクラスがこのカードのクラスと一致する場合のみ適用する。）

効果（英語原文）

[Class Bonus] True Sight (Attacks using this weapon can target units with stealth. Apply this effect only if your champion's class matches this card's class.)

フレーバーテキスト

Before this blade, nothing remains hidden.

このカードの用語

  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • トゥルーサイト（True Sight）この武器を使う攻撃は、ステルスを持つユニットも対象にできる。
  • ステルス（Stealth）このオブジェクトは、トゥルーサイトを持つ攻撃でない限り、攻撃の対象にならない。

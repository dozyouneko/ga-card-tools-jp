効果（日本語）

このターン、対象のユニットの次の攻撃は+2[パワー]を得る。クラスボーナス： そのユニットは、ターン終了時までさらにレンジド2を得る。（そのユニットがディスタントである限り、そのユニットの攻撃は+2[パワー]を得る。この追加効果は、あなたのチャンピオンのクラスがこのカードのクラスと一致する場合のみ適用する。）

効果（英語原文）

Target unit's next attack this turn gets +2 [POWER]. Class Bonus: That unit also gains ranged 2 until end of turn. (As long as that unit is distant, that unit's attacks get +2 [POWER]. Apply the additional effect only if your champion's class matches this card's class.)

このカードの用語

  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • ディスタント（distant）ディスタントはそのユニットの状態を指す。この状態を参照する効果により追加で様々な強化が発生する。この状態は自分のターン終了時に失われる。（相手のターン終了時ではこの状態は失われない）
  • レンジド（Ranged N）そのユニットがディスタントである限り、その攻撃は+Nパワーを得る。

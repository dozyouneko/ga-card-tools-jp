効果（英語原文）

This card costs 1 less to activate if you control a Sword weapon.

Prevent the next 3 combat damage that would be dealt to target unit this turn.

When masterfully used, a sword can deflect any attack with minimal force.

公式裁定（2件）

※裁定は英語原文です。

2022-12-27

Deflecting Edge can only prevent combat damage, which is damage dealt by attacks.

2022-12-27

The damage prevented does not need to be from one source. E.g. three different sources each dealing 1 combat damage each can all be prevented in the same turn.

効果（日本語）

あなたがアーケインエレメントの神獣アライをコントロールしているかぎり、あなたはこのカードを起動する際、そのエレメント条件を無視する。

1つ選ぶ——
・エンパワー4する。その後、「稲妻の掌握」を追放する。
・ターン終了時まで、対象のアライは＋4パワーを得る。その後、「稲妻の掌握」を追放する。

効果（英語原文）

As long as you control an arcane element Shenju ally, ignore this card's elemental requirements as you activate it.

Choose one—
• Empower 4. Then banish Harness Lightning.
• Target ally gets +4 [POWER] until end of turn. Then banish Harness Lightning.

このカードの用語

  • 追放（Banish）カードをゲームから取り除く一時的・恒久的な領域。
  • エンパワー（Empower N）このターン次に起動するスペルを、チャンピオンが+Nレベルを得たものとして起動・解決する。

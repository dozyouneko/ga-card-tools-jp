効果（日本語）

対象のアライにデバフカウンターが乗っている場合、そのアライを破壊する。そうでない場合、それにデバフカウンターを1個置く。

フローティングメモリー

効果（英語原文）

If target ally has a debuff counter on it, destroy that ally. Otherwise, put a debuff counter on it.

Floating Memory

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。
  • デバフカウンター（debuff counter）アライは自身に置かれたデバフカウンター1個につき-1パワーと-1ライフを得る。バフカウンターの逆。

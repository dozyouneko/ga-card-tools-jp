効果（日本語）

アライリンク （このオブジェクトは、対象のアライにリンクした状態で場に出る。リンクが解除された場合、このオブジェクトをサクリファイスする。）

リンクされたアライは+1[ライフ]を得る。

クラスボーナス： そのアライは、反撃している限りさらに+2[パワー]を得る。（追加効果は、あなたのチャンピオンのクラスがこのカードのクラスと一致する場合にのみ適用する。）

効果（英語原文）

Ally Link (This object enters the field linked to target ally. If the link is broken, sacrifice this object.)

Linked ally gets +1 [LIFE]. Class Bonus: That ally also gets +2 [POWER] as long as they're retaliating. (Apply the additional effect only if your champion's class matches this card's class.)

このカードの用語

  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • サクリファイス（Sacrifice）自分のオブジェクトをコストとして墓地へ送る行為。
  • アライリンク（Ally Link）登場時に自分のアライ1体とリンクし、そのアライを参照する効果を持つファンタジア。
  • リンク（Link）対象のオブジェクトをリンク先として場に出る能力。リンク先を失うと、このオブジェクトはサクリファイスされる。リンク先の種類に応じて「ユニットリンク」「チャンピオンリンク」「ウェポンリンク」などと表記される。

効果（日本語）

対象のユニットはディスタントになる。そのユニットが防御中なら、戦闘フェイズを終了する。（フェイズが終了する際、エフェクトスタックにあるすべてのトリガー、起動、マテリアライズを追放する。）

[クラスボーナス] フローティングメモリー

効果（英語原文）

Target unit becomes distant. If that unit is defending, end the combat phase. (As a phase ends, banish all triggers, activations, and materializations on the effects stack.)

[Class Bonus] Floating Memory

このカードの用語

  • 追放（Banish）カードをゲームから取り除く一時的・恒久的な領域。
  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • ディスタント（distant）ディスタントはそのユニットの状態を指す。この状態を参照する効果により追加で様々な強化が発生する。この状態は自分のターン終了時に失われる。（相手のターン終了時ではこの状態は失われない）
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。

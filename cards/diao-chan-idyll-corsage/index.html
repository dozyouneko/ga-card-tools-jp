効果（日本語）

ダオ・チャン・リネージュ

登場時： 好きな数の非チャンピオンのオブジェクトを選び、それぞれにウィザーカウンターを1個置く。

対戦相手がコントロールする非トークンのオブジェクトが破壊されるたび、あなたはそれを追放してよい。そうしたなら、その対戦相手は花のつぼみ・トークンを1体召喚する。

効果（英語原文）

Diao Chan Lineage

On Enter: Choose any amount of non-champion objects and put a wither counter on each of them.

Whenever a non-token object an opponent controls is destroyed, you may banish it. If you do, that opponent summons a Flowerbud token.

このカードの用語

  • 召喚（Summon）指定されたトークンを場に出す。
  • 追放（Banish）カードをゲームから取り除く一時的・恒久的な領域。
  • 登場時（On Enter）そのカードが場に出たときに誘発する効果。
  • リネージュ（Lineage）一番上にある現在のチャンピオンの下に重ねている全てのカード群を指す。
  • ウィザーカウンター（wither counter）各プレイヤーのメインフェイズ開始時、ウィザーカウンターが置かれたオブジェクトは、その1個につき(1)を支払わなければサクリファイスされる（支払えば取り除かれる）。ダオ・チャン系の主要ギミック。

公式裁定（3件）

※裁定は英語原文です。

2025-03-02

The destroyed object must be banished by Diao Chan's trigger from the graveyard to have its controller summon a Flowerbud token. A Flowerbud token will not be summoned on the destruction of a regalia, which never exists in the graveyard. In addition, if the destroyed card was banished before Diao Chan's trigger resolves, because this trigger's effect did not do the banishing, a Flowerbud token will also not be summoned.

2025-03-02

Returning from a combat phase to main phase does not start a new main phase but resumes the previous one. The wither trigger does not happen when returning to main phase from combat.

2025-03-02

The main phase trigger for wither counters is one collective trigger. In one resolution, the turn player decides to pay or not pay for objects with wither counters on them, sacrifices all objects where the all wither counters were not paid for, and then remove all wither counters from all objects they control.

効果（日本語）

最大1体の対象のアライは、ターン終了時まで−1[パワー]を得る。そのアライがオートマトンなら、代わりにそれにデバフカウンターを2個置く。（アライは、自身に置かれたデバフカウンター1個につき−1[パワー]と−1[ライフ]を得る。）

フローティングメモリー （メモリーコストの支払い中に、このカードを墓地から追放してそのコストの1点分の支払いに充ててよい。）

効果（英語原文）

Up to one target ally gets -1 [POWER] until end of turn. If that ally is an Automaton, put two debuff counters on it instead. (Allies get -1 power and -1 life for each debuff counter on them.)

Floating Memory

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。
  • デバフカウンター（debuff counter）アライは自身に置かれたデバフカウンター1個につき-1パワーと-1ライフを得る。バフカウンターの逆。

効果（日本語）

登場時：メモリーコスト0の対象のレガリアを破壊する。

［レベル3以上］「ヴァルッカの侍者」は＋3パワーを得る。

効果（英語原文）

On Enter: Destroy target regalia with memory cost 0.

[Level 3+] Varuckan Acolyte gets +3 [POWER].

フレーバーテキスト

Some of the cabalistic rituals practiced among the mad cultists require material sacrifice.

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • レガリアカードタイプの一つ。マテリアルフェイズや効果によりマテリアルデッキよりマテリアライズされる。
  • レベルチャンピオンの成長段階。レベルを上げて強力な形態にできる。
  • 登場時（On Enter）そのカードが場に出たときに誘発する効果。

公式裁定（1件）

※裁定は英語原文です。

2023-08-09

Varuckan Acolyte's On Enter ability does not specifically target a regalia that you don't control. This means that if you control the only regalia with memory cost 0, that regalia must be the target for this ability.

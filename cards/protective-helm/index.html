効果（日本語）

ユニットリンク （このオブジェクトは対象のユニットにリンクした状態で場に出る。リンクが切れた場合、このオブジェクトをサクリファイスする。）

リンクされたユニットにディスタントの発生源からダメージが与えられる場合、そのダメージのうち1点を軽減する。

効果（英語原文）

Unit Link (This object enters the field linked to target unit. If the link is broken, sacrifice this object.)
If damage would be dealt to linked unit from a distant source, prevent 1 of that damage.

このカードの用語

  • ディスタント（distant）ディスタントはそのユニットの状態を指す。この状態を参照する効果により追加で様々な強化が発生する。この状態は自分のターン終了時に失われる。（相手のターン終了時ではこの状態は失われない）
  • サクリファイス（Sacrifice）自分のオブジェクトをコストとして墓地へ送る行為。
  • リンク（Link）対象のオブジェクトをリンク先として場に出る能力。リンク先を失うと、このオブジェクトはサクリファイスされる。リンク先の種類に応じて「ユニットリンク」「チャンピオンリンク」「ウェポンリンク」などと表記される。

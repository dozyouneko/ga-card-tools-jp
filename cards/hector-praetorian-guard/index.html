効果（日本語）

インビュー3 （このカードを起動する際、リザーブしたすべてのカードを公開してよい。うち3枚以上がネオスエレメントなら、このカードはインビューされた状態になる。）

[クラスボーナス] 登場時： 「ヘクター」がインビューされているなら、オートマトン・ドローン・トークンを2体召喚する。

[クラスボーナス] あなたがコントロールする1体以上の非トークンのネオスエレメントのユニットにダメージが与えられる場合、そのダメージのうちX点を軽減する。Xは、あなたがコントロールするトークンの数である。この置換効果はターンに1回のみ適用する。

効果（英語原文）

Imbue 3

[Class Bonus] On Enter: If Hector is imbued, summon two Automaton Drone tokens.

[Class Bonus] If damage would be dealt to one or more non-token neos element units you control, prevent X of that damage where X is the amount of tokens you control. Apply this replacement effect only once per turn.

このカードの用語

  • 召喚（Summon）指定されたトークンを場に出す。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • 登場時（On Enter）そのカードが場に出たときに誘発する効果。
  • インビュー（Imbue N）起動時にリザーブしたカードを全て公開してよい。うちN枚以上が指定エレメントならインビュー状態になり追加効果を得る。

効果（日本語）

クラスロック （このカードは、あなたのチャンピオンのクラスがこのカードのクラスと一致する場合にのみプレイできる。）

(2)：対象のレンジャーのユニットはディスタントになる。これがこの能力が3回目に解決された機会である場合、カードを1枚あなたのメモリーに引く。この能力は3回のみ起動できる。

効果（英語原文）

Class Locked (Play this card only if your champion’s class matches this card’s class.)

(2): Target Ranger unit becomes distant. If this is the third time this ability has resolved, draw a card into your memory. Activate this ability only thrice.

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • ディスタント（distant）ディスタントはそのユニットの状態を指す。この状態を参照する効果により追加で様々な強化が発生する。この状態は自分のターン終了時に失われる。（相手のターン終了時ではこの状態は失われない）
  • クラスロック（Class Locked）あなたのチャンピオンのクラスがこのカードのクラスと一致していなければ、このカードは起動できない。

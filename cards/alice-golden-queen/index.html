効果（日本語）

継承効果 → あなたが起動するチェスマン・コマンド・アタックカードは、+1[パワー]を得た状態でインテントに入る。

リネージュ・リリース → あなたがコントロールするアウェイクのチェスマン・アライそれぞれについて、このターン、そのアライに次に与えられる3点のダメージを軽減する。

効果（英語原文）

Inherited Effect — Chessman Command attack cards you activate enter the intent with +1[POWER].

Lineage Release — For each awake Chessman ally you control, prevent the next 3 damage that would be dealt to that ally this turn.

このカードの用語

  • 継承効果（Inherited Effect）下位レベルから引き継がれ、レベルアップ後も有効なチャンピオンの効果。
  • リネージュ（Lineage）一番上にある現在のチャンピオンの下に重ねている全てのカード群を指す。

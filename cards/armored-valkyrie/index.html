効果（英語原文）

Steadfast (This ally can retaliate while rested and doesn't rest to do so.)

[Class Bonus] As long as Armored Valkyrie is retaliating, it gets +2 [POWER]. (Apply this effect only if your champion's class matches this card's class.)

効果（日本語）

[レベル1+] このカードの起動コストが3少なくなる。

カーディストリ → (7)：このカードは「あなたがコントロールする他のスーテッド・アライは+1[パワー]を得て、ヴィガーを持つ。」を得る。この能力は、あなたがコントロールする、異なるリザーブコストを持つスーテッド・オブジェクト1つにつき起動コストが(1)少なくなる。この能力は1回のみ起動できる。

効果（英語原文）

[Level 1+] This card costs 3 less to activate.

Cardistry — (7): Chance, Seven of Spades gains "Other Suited allies you control get +1 [POWER] and have vigor." This ability costs (1) less to activate for each Suited object you control with different reserve costs. Activate this ability only once.

このカードの用語

  • リザーブコストの支払いとして、手札からメモリー領域へ移すこと。リザーブコストはこの方法で支払う。
  • レベルチャンピオンの成長段階。レベルを上げて強力な形態にできる。
  • ヴィガー（Vigor）自分のエンドフェイズの開始時に、このユニットをアウェイクにする。
  • カーディストリ（Cardistry）スート系のオブジェクトの数だけコストが下がる起動型能力。1回のみ起動できる。

2025-12-04

The "activate this ability only once" applies to each instance of a Cardistry ability on a card. If you have multiple copies of a card with the same Cardistry ability in play, each one can activate its ability one time.

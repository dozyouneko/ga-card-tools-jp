効果（日本語）

チャンピオンリンク （このオブジェクトは、対象のチャンピオンにリンクした状態で場に出る。リンクが解除された場合、このオブジェクトをサクリファイスする。）

リンクされたチャンピオンが各ターンに行う最初の攻撃は+2[パワー]を得る。クラスボーナス： そのチャンピオンはさらに、あなたの墓地にあるユニークのアライカード1枚につき+2[ライフ]を得る。

効果（英語原文）

Champion Link (This object enters the field linked to target champion. If the link is broken, sacrifice this object.)

Linked champion's first attack each turn gets +2 [POWER]. Class Bonus: That champion also gets +2 [LIFE] for each unique ally card in your graveyard.

このカードの用語

  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • 墓地（Graveyard）破棄・使用済みのカードが置かれる領域。
  • サクリファイス（Sacrifice）自分のオブジェクトをコストとして墓地へ送る行為。
  • リンク（Link）対象のオブジェクトをリンク先として場に出る能力。リンク先を失うと、このオブジェクトはサクリファイスされる。リンク先の種類に応じて「ユニットリンク」「チャンピオンリンク」「ウェポンリンク」などと表記される。

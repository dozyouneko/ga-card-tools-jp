効果（日本語）

[レスト]：いずれかの墓地からカードを1枚選び、それを追放する。そのカードがフローティングメモリーを持つ場合、「マーリン、記憶泥棒」にレベルカウンターを1個置く。（チャンピオンは、自身に置かれたレベルカウンター1個につき+1レベルを得る。）

効果（英語原文）

[REST]: Choose a card from a graveyard and banish it. If that card has floating memory, put a level counter on Merlin, Memory Thief. (Champions get +1 level for each level counter on them.)

フレーバーテキスト

"Just forget. It will be easier for you."

このカードの用語

  • 追放（Banish）カードをゲームから取り除く一時的・恒久的な領域。
  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • レベルチャンピオンの成長段階。レベルを上げて強力な形態にできる。
  • 墓地（Graveyard）破棄・使用済みのカードが置かれる領域。
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。
  • レベルカウンター（Level Counter）チャンピオンに置かれ、置かれている数1個につき+1レベルを得るカウンター。

公式裁定（1件）

※裁定は英語原文です。

2022-12-27

If Merlin's ability banishes a card that has Floating Memory but is restricted by a Class Bonus, a level counter will only be placed on Merlin if that card's owner has that Class Bonus restriction enabled.

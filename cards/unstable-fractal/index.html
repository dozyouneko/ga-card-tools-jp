効果（日本語）

ハインダード （このオブジェクトはレスト状態で場に出る。） リザーブ可能 （リザーブコストの支払い中に、このオブジェクトをレストして、そのコストの(1)分を支払ってよい。）

[クラスボーナス] (3)、[レスト]、「不安定なフラクタル」をサクリファイスする：スペルとして、メモリーコストが0、またはリザーブコストが5以下の対象のアイテムを破壊する。

効果（英語原文）

Hindered (This object enters the field rested.)

Reservable

[Class Bonus] (3), [REST], Sacrifice Unstable Fractal: As a Spell, destroy target item with memory cost 0 or reserve cost 5 or less.

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • リザーブコストの支払いとして、手札からメモリー領域へ移すこと。リザーブコストはこの方法で支払う。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • サクリファイス（Sacrifice）自分のオブジェクトをコストとして墓地へ送る行為。
  • ハインダード（Hindered）このオブジェクトはレスト状態で場に出る。
  • リザーブ可能（Reservable）リザーブコストの支払い中に、このオブジェクトをレストして、そのコストの(1)分を支払える。

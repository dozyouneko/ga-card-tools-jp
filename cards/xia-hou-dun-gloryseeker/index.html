効果（日本語）

レンジド2 （このユニットがディスタントである限り、その攻撃は+2[パワー]を得る。）

[クラスボーナス] あなたが剣または弓のウェポンをコントロールしている限り、「シャホウ・ドゥン」は+1[パワー]を得る。（この効果は、あなたのチャンピオンのクラスがこのカードのクラスと一致する場合にのみ適用する。）

効果（英語原文）

Ranged 2 (As long as this unit is distant, its attacks get +2 [POWER].)

[Class Bonus] As long as you control a Sword or Bow weapon, Xia Hou Dun, Gloryseeker gets +1 [POWER]. (Apply this effect only if your champion's class matches this card's class.)

このカードの用語

  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • ディスタント（distant）ディスタントはそのユニットの状態を指す。この状態を参照する効果により追加で様々な強化が発生する。この状態は自分のターン終了時に失われる。（相手のターン終了時ではこの状態は失われない）
  • レンジド（Ranged N）そのユニットがディスタントである限り、その攻撃は+Nパワーを得る。

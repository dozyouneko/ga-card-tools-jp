効果（日本語）

[クラスボーナス] [メモリー4+] グリンプス3を行う。（この効果は、あなたのチャンピオンのクラスがこのカードのクラスと一致し、かつあなたのメモリーにカードが4枚以上ある場合にのみ適用する。）

カードを1枚あなたのメモリーに引く。

効果（英語原文）

[Class Bonus] [Memory 4+] Glimpse 3. (Apply this effect only if your champion's class matches this card's class and only if there are four or more cards in your memory.)

Draw a card into your memory.

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • グリンプス（Glimpse）自分のデッキの上からX枚を見て、任意の順で好きなように上／下に戻す占術系のキーワード。

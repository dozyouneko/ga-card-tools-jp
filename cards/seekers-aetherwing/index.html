効果（日本語）

効果（英語原文）

(Aetherwing — Must be loaded to use for an attack and can’t be used with an attack card.)

[Class Bonus] Spellshroud (This object can't be targeted by Spells.)

[Class Bonus] True Sight (Attacks using this weapon can target units with stealth.)

このカードの用語

  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • スペルシュラウド（Spellshroud）このオブジェクトはスペルの対象にできない。
  • エーテルウィング（Aetherwing）エーテルチャージ等を装填して運用する武器を表すカードサブタイプ。関連効果の条件・参照に使われる。
  • 装填（load）カードを武器などに仕込み、後で発射・消費するための配置。
  • トゥルーサイト（True Sight）この武器を使う攻撃は、ステルスを持つユニットも対象にできる。
  • ステルス（Stealth）このオブジェクトは、トゥルーサイトを持つ攻撃でない限り、攻撃の対象にならない。

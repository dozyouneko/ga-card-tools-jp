効果（日本語）

フローティングメモリーを持つカードがプレイヤーの墓地に置かれる場合、代わりにそのプレイヤーはそれを追放する。

効果（英語原文）

If a card with floating memory would be put into a player's graveyard, that player banishes it instead.

このカードの用語

  • 追放（Banish）カードをゲームから取り除く一時的・恒久的な領域。
  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • 墓地（Graveyard）破棄・使用済みのカードが置かれる領域。
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。

公式裁定（2件）

※裁定は英語原文です。

2025-03-02

Brackish Lutist's ability is a replacement effect that can change how cards resolve mid-resolution. The cards that are instead banished would have never hit the graveyard.

2025-03-02

If a card that has Floating Memory would be sent to the graveyard but restricted by a class bonus, it will only be banished if that card's owner has that class bonus restriction enabled.

効果（日本語）

[クラスボーナス] 登場時： メモリーコストが0の弾丸カードを1枚、あなたのマテリアルデッキからマテリアライズする。（この効果は、あなたのチャンピオンのクラスがこのカードのクラスと一致する場合にのみ適用する。）

効果（英語原文）

[Class Bonus] On Enter: Materialize a Bullet card with memory cost 0 from your material deck. (Apply this effect only if your champion’s class matches this card’s class.)

フレーバーテキスト

Preloaded with whatever you deem necessary.

このカードの用語

  • マテリアライズマテリアルデッキのカードを、そのコストを支払って場に出す行為。
  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • 登場時（On Enter）そのカードが場に出たときに誘発する効果。

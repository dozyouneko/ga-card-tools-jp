効果（日本語）

［クラスボーナス］ステルス （このユニットは、トゥルーサイトによって許可されないかぎり、攻撃の対象にできない。）

登場時：対象の対戦相手のメモリーを見て、その後任意のカード名を1つ選ぶ。

選ばれた名前を持つカードはプレイコストが1多くなる。

効果（英語原文）

[Class Bonus] Stealth (This unit can't be targeted by attacks unless permitted by true sight.)

On Enter: Look at target opponent's memory, then choose any card name.

Cards with the chosen name cost 1 more to play.

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • 登場時（On Enter）そのカードが場に出たときに誘発する効果。
  • トゥルーサイト（True Sight）この武器を使う攻撃は、ステルスを持つユニットも対象にできる。
  • ステルス（Stealth）このオブジェクトは、トゥルーサイトを持つ攻撃でない限り、攻撃の対象にならない。

公式裁定（4件）

※裁定は英語原文です。

2022-12-27

The type of cost increased is based on the card's default cost type. If the affected card has a reserve costs, it costs 1 more reserve. If the affected card has a memory cost, it costs 1 more memory.

2022-12-27

You do not need to name a card that you see revealed from the opponent's memory.

2022-12-27

Nia's cost increasing effect affects all players.

2022-12-27

The cost increase only affects cards that are played, which is a term used to group both activate and materialize.

効果（日本語）

効果（英語原文）

[Class Bonus] Ranged 2 (As long as this unit is distant, its attacks get +2 [POWER]. Apply this effect only if your champion's class matches this card's class.)

フレーバーテキスト

Military-trained infantry able to adjust position and sight an enemy across the battlefield.

このカードの用語

  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • ディスタント（distant）ディスタントはそのユニットの状態を指す。この状態を参照する効果により追加で様々な強化が発生する。この状態は自分のターン終了時に失われる。（相手のターン終了時ではこの状態は失われない）
  • レンジド（Ranged N）そのユニットがディスタントである限り、その攻撃は+Nパワーを得る。

効果（日本語）

コマンドウィル1 （このユニットがコマンドカードを使って攻撃している限り、+1[パワー]を得る。）

[アリスボーナス] アライキル時： キルされたユニットがインターセプトまたはタウントを持っていたなら、このカードをアウェイクにする。この能力は各ターンに1回のみ誘発する。

効果（英語原文）

Commanded Will 1 (As long as this unit is attacking using a Command card, it gets +1 [POWER].)

[Alice Bonus] On Ally Kill: If the killed unit had intercept or taunt, wake up Rowland, Schwartz Knight. Trigger this ability only once per turn.

このカードの用語

  • タウント（taunt）アウェイクの間、相手の攻撃宣言時に、可能なら他のオブジェクトより先に対象にされなければならない。ステルスなどにより対象に取れない場合、他のオブジェクトを対象に取ることができる。
  • インターセプト（Intercept）このアライがアウェイクの間に自分のチャンピオンが攻撃されたとき、対象をこのアライに変更してもよい。
  • コマンドウィル（Commanded Will N）コマンドカードを使って攻撃している限り、+Nパワーを得る。

効果（日本語）

タウント （アウェイクの間、可能ならこのアライは対戦相手の攻撃宣言時にあなたがコントロールする他のオブジェクトより先に対象にされなければならない。）

「防水の軍曹」にダメージが与えられる場合、そのダメージを軽減する。この置換効果は各ターンに1回のみ適用する。

効果（英語原文）

Taunt (While awake, this ally must be targeted before other objects you control during your opponents’ attack declarations if able.)

If damage would be dealt to Floodward Sergeant, prevent that damage. Apply this replacement effect only once each turn.

このカードの用語

  • タウント（taunt）アウェイクの間、相手の攻撃宣言時に、可能なら他のオブジェクトより先に対象にされなければならない。ステルスなどにより対象に取れない場合、他のオブジェクトを対象に取ることができる。

効果（日本語）

[クラスボーナス] 死亡時： パワーセル・トークンを1体召喚する。（この効果は、あなたのチャンピオンのクラスがこのカードのクラスと一致する場合にのみ適用する。）

効果（英語原文）

[Class Bonus] On Death: Summon a Powercell token. (Apply this effect only if your champion’s class matches this card’s class.)

フレーバーテキスト

“Tired of pesky pirates? These portside units run on self-powered fuel cells that are fully waterproof!” — Gearstride Automatics Advertisement

このカードの用語

  • 召喚（Summon）指定されたトークンを場に出す。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • 死亡時（On Death）このオブジェクトが死亡したときに誘発する効果の見出し。

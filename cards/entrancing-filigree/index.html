魅惑の装飾細工Entrancing Filigree

効果（日本語）

登場時： スペルとして、あなたがコントロールしていない対象の非チャンピオンのオブジェクトを追放する。（スペルの能力は、スペルシュラウドを持つオブジェクトを対象にできない。）

退場時： 追放されたオブジェクトを、オーナーのコントロール下でレスト状態で場に戻す。

効果（英語原文）

On Enter: As a Spell, banish target non-champion object you don't control. (Spell abilities can't target objects with spellshroud.)

On Leave: Return the banished object to the field under its owner's control rested.

このカードの用語

  • 追放（Banish）カードをゲームから取り除く一時的・恒久的な領域。
  • 登場時（On Enter）そのカードが場に出たときに誘発する効果。
  • スペルシュラウド（Spellshroud）このオブジェクトはスペルの対象にできない。
  • 退場時（On Leave）このオブジェクトが場を離れたときに誘発する効果。

効果（日本語）

インターセプト、トゥルーサイト、ヴィガー

あなたがコントロールするチャンピオンはスペルシュラウドを持つ。

あなたがコントロールする別のクラックスエレメントのユニットがダメージを受ける場合、そのダメージの半分（端数切り上げ）を軽減する。

効果（英語原文）

Intercept, True Sight, Vigor

Champions you control have spellshroud.

If another crux element unit you control would take damage, prevent half of that damage, rounded up.

このカードの用語

  • スペルシュラウド（Spellshroud）このオブジェクトはスペルの対象にできない。
  • インターセプト（Intercept）このアライがアウェイクの間に自分のチャンピオンが攻撃されたとき、対象をこのアライに変更してもよい。
  • トゥルーサイト（True Sight）この武器を使う攻撃は、ステルスを持つユニットも対象にできる。
  • ヴィガー（Vigor）自分のエンドフェイズの開始時に、このユニットをアウェイクにする。

公式裁定（1件）

※裁定は英語原文です。

2023-08-09

Because The Majestic Spirit is a regalia, if it would be sent to the deck, memory, or hand, it is sent to its owner's material deck instead. If it would die, it is banished instead (On Kill effects won't trigger).

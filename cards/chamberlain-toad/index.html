効果（日本語）

（エグザルテッド → 他のアドバンスドエレメントが有効である限り、このエレメントもあなたにとって有効になる。）

[クラスボーナス] (2)：このカードをアウェイクにし、それはターン終了時までタウントを得る。この能力は対戦相手のリコレクションフェイズ中のみ起動できる。

[クラスボーナス] フローティングメモリー

効果（英語原文）

(Exalted — This element is enabled for you as long as you have another advanced element enabled.)

[Class Bonus] (2): Wake up Chamberlain Toad and it gains taunt until end of turn. Activate this ability only during an opponent's recollection phase.

[Class Bonus] Floating Memory

このカードの用語

  • メモリー伏せて管理する専用領域。多くのカードがここを経由して起動される。
  • クラスボーナス自分のチャンピオンのクラスが一致するとき追加効果／コスト軽減を得る。
  • フローティングメモリー（Floating Memory）メモリーコスト支払い中に、このカードを墓地から追放してコストの一部に充てられる。
  • タウント（taunt）アウェイクの間、相手の攻撃宣言時に、可能なら他のオブジェクトより先に対象にされなければならない。ステルスなどにより対象に取れない場合、他のオブジェクトを対象に取ることができる。
  • エグザルテッド（Exalted）他のアドバンスドエレメントを有効にしている限り、このエレメントも有効になる。
  • アドバンスドエレメント（advanced element）基本エレメント（ノーム・火・水・風）を除くエレメント（ラクサム・アンブラ・テラ・アストラ・クラックス・アーケイン・エクシア・ネオス・エグザルテッド）の総称。

効果（日本語）

このカードを追放する： カードを1枚引く。この能力は、対戦相手が風エレメントのチャンピオンをコントロールしている場合のみ起動できる。

効果（英語原文）

Banish Wind Resonance Bauble: Draw a card. Activate this ability only if an opponent controls a wind element champion.

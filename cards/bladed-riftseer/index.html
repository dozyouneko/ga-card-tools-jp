効果（日本語）

[レベル1+] トゥルーサイト （トゥルーサイトを持つユニットは、ステルスを持つオブジェクトを攻撃できる。この効果は、あなたのチャンピオンがレベル1以上の場合のみ適用する。）

効果（英語原文）

[Level 1+] True Sight (Units with true sight can attack objects with stealth. Apply this effect only if your champion is level 1 or higher.)

このカードの用語

  • レベルチャンピオンの成長段階。レベルを上げて強力な形態にできる。
  • トゥルーサイト（True Sight）この武器を使う攻撃は、ステルスを持つユニットも対象にできる。
  • ステルス（Stealth）このオブジェクトは、トゥルーサイトを持つ攻撃でない限り、攻撃の対象にならない。
